# Sky Whales Demo
Pitch demo for whale-based vehicular combat game

# Controls
## Keyboard
- Turn: Arrow keys or WASD
- Accelerate: Left shift
- Brake: Left control
- Fire: Space
## Gamepad
- Turn: Left stick
- Accelerate: A
- Brake: B
- Fire: X
